# IOP_FEV1_ECG

All code used in development of an algorithm to extract FEV1 and FEV1 z-score from ECG signals recorded from the fingers.
Run the IOP_Focus_Issue_FEV1 Estimation Using Extracted Features.ipynb for a quick example. Fake FEV1, FVC and BMI values have been used. Change these with the appropriate values for your data.
